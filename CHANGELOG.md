### Added
- Initial project structure
- mechano metapackage

## [0.1.0] - 2024-01-01

### Added
- Initial release
- Basic differential drive robot model
- ros2_control hardware interface for motor control
- Nav2 integration for autonomous navigation
- SLAM support with slam_toolbox
- Gazebo simulation
- Teleop support (keyboard and joystick)
